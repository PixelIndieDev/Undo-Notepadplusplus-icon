# Undo Notepad++ file icon
A registry script that changes the Notepad++ file type icon to the default Windows text document icon.

I created this `.reg` file to quickly replace the default Notepad++ file icon with the standard Windows text document icon, as I prefer that look for my text files over the Notepad++ file icon.

> [!NOTE]
> This tweak changes the icon globally for **all** file types currently associated with Notepad++. I have not found a way to set individual file type icons for files that are currently associated with Notepad++.

## How to use
1. **Download** the UndoNotepadplusplusIcon.reg file
2. **Double-click the file** to apply the changes to your Windows Registry
3. **Restart File Explorer** (or restart your PC) for the changes to take effect
4. **Done**

### How to Restart File Explorer
1. Press `Ctrl + Shift + Esc` to open **Task Manager**.
2. Locate **Windows Explorer** in the processes list.
3. Right-click it and select **Restart**.
